  ПОЛНАЯ ИНСТРУКЦИЯ ПО УСТАНОВКЕ СКРИПТА В GOOGLE ТАБЛИЦУ
═══════════════════════════════════════════════════════════════

ШАГ 1: ОТКРЫТЬ GOOGLE APPS SCRIPT
───────────────────────────────────────────────────────────────
1. Откройте вашу Google Таблицу с акциями
2. В меню выберите: Расширения → Apps Script
3. Откроется редактор Apps Script

ШАГ 2: ВСТАВИТЬ СКРИПТ
───────────────────────────────────────────────────────────────
1. В редакторе Apps Script выделите ВЕСЬ существующий код (Ctrl+A или Cmd+A)
2. Удалите его (Delete)
3. Откройте файл GOOGLE_APPS_SCRIPT_FULL.js из проекта
4. Скопируйте ВЕСЬ код из файла (Ctrl+A, Ctrl+C)
5. Вставьте в редактор Apps Script (Ctrl+V)
6. Сохраните: Ctrl+S (или Cmd+S на Mac)

ШАГ 3: РАЗВЕРНУТЬ КАК ВЕБ-ПРИЛОЖЕНИЕ
───────────────────────────────────────────────────────────────
1. Нажмите кнопку "Развернуть" (Deploy) в правом верхнем углу
2. Выберите "Новое развертывание" (New deployment)
3. Нажмите на иконку шестеренки ⚙️ рядом с "Выберите тип"
4. Выберите "Веб-приложение" (Web app)
5. В поле "У кого есть доступ" выберите "Все" (Anyone)
6. Нажмите "Развернуть" (Deploy)

ШАГ 4: СКОПИРОВАТЬ URL
───────────────────────────────────────────────────────────────
1. После развертывания появится окно с URL
2. Нажмите кнопку "Copy" рядом с URL
3. URL будет вида: https://script.google.com/macros/s/.../exec
4. Сохраните этот URL

ШАГ 5: ОБНОВИТЬ menu.html (УЖЕ СДЕЛАНО!)
───────────────────────────────────────────────────────────────
✅ URL уже вставлен в menu.html автоматически!
✅ Файл menu.html уже обновлен!

ШАГ 6: ПРОВЕРИТЬ РАБОТУ
───────────────────────────────────────────────────────────────
1. Откройте скопированный URL в браузере
2. Вы должны увидеть JSON с активными акциями
3. Если видите JSON - всё работает! ✅
