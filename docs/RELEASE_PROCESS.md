# RELEASE_PROCESS

Процесс релиза:

1. Обновить `CHANGELOG.md` в секции Unreleased.
2. Обновить версию (семантическое версионирование).
3. Создать тег `vX.Y.Z` и сделать release.
4. Проверить CI и выполнить деплой (если применимо).
