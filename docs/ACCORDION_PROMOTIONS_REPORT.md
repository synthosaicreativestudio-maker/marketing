## ✅ Реализованный функционал

### **1. Уведомления в чат** ✅

#### **Как работает:**
1. Администратор меняет статус акции на "Опубликовано" в Google Sheets
2. Бот каждые 30 секунд проверяет новые акции
3. Отправляет уведомление всем авторизованным пользователям

#### **Содержание уведомления:**
```
🎉 Новая акция опубликована!

📢 Название акции
📅 Действует: 01.01.2025 - 31.01.2025

Описание акции (до 200 символов)...

👀 Посмотреть подробнее ↓
[👀 Ознакомиться подробнее] <- Кнопка
```

#### **Технические детали:**
- **Кнопка**: InlineKeyboardButton (не ReplyKeyboardMarkup)
- **URL**: Добавлен параметр `?section=promotions` для прямого перехода
- **Текст кнопки**: "👀 Ознакомиться подробнее"
- **Получатели**: Все авторизованные пользователи

### **2. Прямой переход в раздел акций** ✅

#### **Как работает:**
1. Пользователь нажимает кнопку в уведомлении
2. Открывается мини-приложение с параметром `?section=promotions`
3. Автоматически происходит переход в раздел акций

#### **Технические детали:**
```javascript
// Проверяем URL параметры для автоматического перехода
const urlParams = new URLSearchParams(window.location.search);
const section = urlParams.get('section');
if (section === 'promotions') {
    // Автоматически переходим в раздел акций
    setTimeout(() => {
        goToPromotions();
    }, 500);
}
```

### **3. Аккордеон в мини-приложении** ✅

#### **Как работает:**
1. Все активные акции отображаются списком
2. Каждая акция - это карточка с заголовком
3. Нажатие на карточку раскрывает/скрывает детали
4. Плавная анимация раскрытия/скрытия

#### **Визуальные элементы:**
- **Заголовок**: Название акции + статус + стрелка ▼/▲
- **Содержимое**: Описание, даты, статус, медиа контент
- **Анимация**: Плавное раскрытие за 0.3 секунды
- **Индикатор**: Стрелка поворачивается при раскрытии

#### **CSS стили:**

## 🔧 Технические детали

### **Новые функции JavaScript:**

#### **`togglePromotion(index)`**
```javascript
function togglePromotion(index) {
    const cards = document.querySelectorAll('.promotion-card');
    const card = cards[index];
    if (card) {
        card.classList.toggle('expanded');
    }
}
```

#### **`renderMediaContent(media)`**
```javascript
function renderMediaContent(media) {
    if (!media || media.length === 0) {
        return '';
    }
    
    let html = '<div class="promotion-media">';
    media.forEach(item => {
        if (item.type === 'image') {
            html += `<div class="media-item"><img src="${item.url}" alt="Медиа контент" loading="lazy"></div>`;
        } else if (item.type === 'video') {
            if (item.url.includes('youtube.com/embed')) {
                html += `<div class="media-item"><iframe src="${item.url}" frameborder="0" allowfullscreen></iframe></div>`;
            } else {
                html += `<div class="media-item"><video controls><source src="${item.url}" type="video/mp4"></video></div>`;
            }
        }
    });
    html += '</div>';
    return html;
}
```
