## 🎯 ОПИСАНИЕ СИСТЕМЫ

Система автоматического ведения документации обеспечивает постоянное обновление всех технических документов, чек-листов, роадмэпов и описаний проекта. Документация обновляется автоматически при каждом значительном изменении в проекте.
