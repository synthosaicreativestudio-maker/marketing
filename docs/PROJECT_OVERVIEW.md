## 🎯 ОПИСАНИЕ ПРОЕКТА

**Marketing Bot** - это интеллектуальный Telegram-бот для управления маркетинговыми кампаниями, акциями и клиентскими взаимодействиями. Проект представляет собой комплексное решение для автоматизации маркетинговых процессов с интеграцией в Google Sheets и современным веб-интерфейсом.

### 🏆 Основные возможности
- **Автоматические уведомления** о новых акциях и мероприятиях
- **Интеграция с Google Sheets** для управления данными
- **Современный веб-интерфейс** через Telegram WebApp
- **Система авторизации** с кэшированием
- **Мониторинг производительности** и обработка ошибок
- **Асинхронная архитектура** для высокой производительности

---

## 🏗️ АРХИТЕКТУРА СИСТЕМЫ

### 🔧 Основные компоненты

#### 1. **Telegram Bot Core** (`bot.py`)
- Основной файл бота с обработчиками команд
- Интеграция с Telegram Bot API
- Управление пользовательскими сессиями
- Обработка WebApp данных

#### 2. **Google Sheets Integration**
- **`async_promotions_client.py`** - асинхронный клиент для акций
- **`sheets_client.py`** - клиент для авторизации и тикетов
- **`promotions_client.py`** - синхронный клиент акций

#### 3. **Web Application** (`spa_menu.html`)
- Telegram WebApp интерфейс
- Аккордеон для отображения акций
- Кэширование на клиенте
- Интеграция с Telegram WebApp API

#### 4. **Система кэширования** (`auth_cache.py`)
- Кэширование статуса авторизации
- Thread-safe операции
- Персистентное хранение

#### 5. **Мониторинг и обработка ошибок**
- **`performance_monitor.py`** - мониторинг производительности
- **`error_handler.py`** - обработка ошибок
- **`network_resilience.py`** - устойчивость к сбоям сети

## 🔄 ЖИЗНЕННЫЙ ЦИКЛ АКЦИЙ

### 1. **Создание акции**
- Администратор создает акцию в Google Sheets
- Устанавливает статус "Активна"
- Заполняет все необходимые поля

### 2. **Мониторинг**
- Бот проверяет новые акции каждые 30 секунд
- Ищет акции со статусом "Активна" и пустой колонкой "Уведомление"

### 3. **Уведомление**
- Бот отправляет уведомление всем авторизованным пользователям
- Помечает уведомление как отправленное в Google Sheets

### 4. **Отображение**
- Пользователи могут просмотреть акции в WebApp
- Аккордеон показывает детальную информацию
- Поддержка медиа контента
