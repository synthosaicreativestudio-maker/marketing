Этот раздел содержит конкретные задачи, проблемы и технический долг.

### 🚀 **Текущие задачи (В процессе)**

- [ ] **Мониторинг производительности:** Отслеживание времени ответа ключевых функций.
- [ ] **Оптимизация кэширования:** Настройка TTL (времени жизни) для разных типов данных.
- [ ] **Улучшение логирования:** Внедрение структурированных логов для более удобного анализа.

### 📋 **Бэклог задач (Планируется)**

#### Архитектура
- [ ] **Рефакторинг класса Bot:** Разбить монолитный класс (1600+ строк) на отдельные сервисы (`AuthService`, `TicketsService`, и т.д.).
- [ ] **Внедрение Dependency Injection:** Уменьшить жесткую связанность компонентов.
- [ ] **Улучшение обработки ошибок:** Заменить общие `except Exception` на обработку конкретных типов исключений.

#### Производительность
- [ ] **Полный переход на async:** Заменить все блокирующие вызовы `run_blocking()` на нативные асинхронные клиенты (aiohttp, asyncpg).
- [ ] **Connection Pooling:** Использовать `aiohttp.ClientSession` для переиспользования HTTP-соединений.

#### Надежность и Безопасность
- [ ] **Circuit Breakers:** Добавить защиту от каскадных сбоев при недоступности внешних API.
- [ ] **Валидация данных:** Внедрить строгую валидацию входных данных (например, с помощью Pydantic).
- [ ] **Rate Limiting:** Добавить ограничение на частоту запросов для защиты от спама.

#### Frontend
- [ ] **Улучшение UI:** Интегрировать CSS-фреймворк (Bootstrap) и обновить дизайн мини-приложений.

#### Новые функции
- [ ] **Автоматические тесты:** Покрыть unit-тестами >80% кодовой базы.
- [ ] **CI/CD:** Настроить автоматическое тестирование и развертывание через GitHub Actions.
- [ ] **Prometheus + Grafana:** Внедрить систему мониторинга для сбора и визуализации метрик.

### 🐛 **Технический долг и известные проблемы**

- **Монолитная архитектура:** Основная логика сосредоточена в `bot.py`.
- **Хардкод конфигурации:** Некоторые настройки зашиты в коде вместо `config.ini`.
- **manage_bot.py status:** Иногда показывает "ОСТАНОВЛЕН" при работающем боте.
- **Event loop errors:** Периодические ошибки при перезапуске бота.
