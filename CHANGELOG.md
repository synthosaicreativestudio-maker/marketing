## [2.0.0] - 2025-09-05

Это крупное обновление, сфокусированное на производительности, стабильности и асинхронности.

### Added
- **Асинхронный клиент для акций:** Добавлен `AsyncPromotionsClient` для неблокирующей работы с Google Sheets.
- **Система кэширования авторизации:** Внедрен `auth_cache.py` для локального кэширования статуса авторизации пользователей, что значительно снизило количество запросов к Google Sheets и ускорило работу бота.
- **Мониторинг производительности:** Добавлен декоратор `@monitor_performance` для отслеживания времени выполнения ключевых функций.
- **Фоновый мониторинг:** Добавлены фоновые задачи для мониторинга ответов специалистов и появления новых акций.

### Changed
- **Переход на асинхронность:** Основные блокирующие операции (работа с `gspread`) теперь выполняются в отдельных потоках через `asyncio.to_thread`, предотвращая "замерзание" бота.
- **Структура `bot.py`:** Проведен частичный рефакторинг, логика разделена на большее количество внутренних методов.

### Fixed
- **Проблема с "замерзанием" бота:** Устранена основная причина медленной работы путем внедрения асинхронных операций.

## [1.5.0] - 2025-09-03

Этот релиз был посвящен исправлению критических ошибок в работе мини-приложения (WebApp) и системы авторизации.

### Added
- **Интерфейс "Аккордеон":** В мини-приложении для акций внедрен интерфейс "аккордеон", позволяющий компактно отображать большое количество акций с медиа-контентом.

### Changed
- **Механизм кнопки авторизации:** Изменен способ вызова окна авторизации для повышения надежности на мобильных устройствах.

### Fixed
- **Критическая ошибка авторизации на iOS:** Исправлена проблема, из-за которой `sendData` не срабатывал в `InlineKeyboardMarkup` на некоторых устройствах. Внедрен гибридный подход с `ReplyKeyboardMarkup`.
- **Некорректная работа кнопок в WebApp:** Исправлены ошибки, при которых кнопки "Уведомить" и "Подробнее" не работали или работали некорректно.

## [1.0.0] - 2025-08-20

Первый стабильный релиз проекта.

### Added
- **Базовый функционал бота:** Обработка команд `/start`, `/menu`.
- **Система авторизации:** Реализована базовая проверка пользователя по коду и телефону через Google Sheets.
- **Система тикетов:** Реализована запись обращений пользователей в Google Sheets.
- **Интеграция с OpenAI:** Добавлена возможность получения ответов от AI-ассистента.
- **Мини-приложение (WebApp):** Создана первая версия личного кабинета.
- **Система уведомлений:** Реализована отправка уведомлений об акциях.
