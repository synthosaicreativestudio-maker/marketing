## 🔧 Возможные проблемы и решения

### 1. WebApp API недоступен
**Симптомы:** `sendData доступен: false`
**Решение:** Автоматически переключается на команду `/auth TEST123 89991234567`

### 2. Данные не доходят до бота
**Симптомы:** Нет логов "Получены данные от WebApp"  
**Решение:** Проверить URL мини-приложения и права доступа

### 3. JSON не парсится
**Симптомы:** Ошибка "Не удалось разобрать JSON"
**Решение:** Проверить формат отправляемых данных

### 4. Пользователь не найден в таблице
**Симптомы:** "Пользователь не найден по коду"
**Решение:** Проверить данные в Google Sheets

## 🚀 Fallback механизм

Если WebApp API не работает:

1. **Автоматически генерируется команда:** `/auth TEST123 89991234567`
2. **Копируется в буфер обмена** (если доступно)
3. **Показывается alert** с командой для ручного копирования
4. **Пользователь вставляет команду** в чат с ботом
5. **Бот обрабатывает через** `auth_command` метод
