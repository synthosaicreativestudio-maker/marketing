# 🚂 Railway - Автоматический деплой бота

## 📋 **Что настроено:**

✅ `railway.json` - конфигурация Railway
✅ `Procfile` - команда запуска
✅ `runtime.txt` - версия Python
✅ `.railwayignore` - исключения

## 🚀 **Как развернуть:**

### 1. **Подключение к Railway:**
- Зайдите на [railway.app](https://railway.app)
- Войдите через GitHub
- Нажмите "New Project"
- Выберите "Deploy from GitHub repo"

### 2. **Выбор репозитория:**
- Найдите ваш `@marketing` репозиторий
- Railway автоматически подключится

### 3. **Настройка переменных:**
В Railway Dashboard добавьте переменные окружения:
```
BOT_TOKEN=ваш_токен_бота
SHEET_URL=ссылка_на_таблицу
OPENAI_API_KEY=ваш_ключ_openai
```

### 4. **Автодеплой:**
- Railway автоматически деплоит при `git push`
- Бот будет работать 24/7
- Автоперезапуск при падении

## ⚡ **Преимущества Railway:**

- 🚀 **Автоматический деплой** из GitHub
- 🔄 **Автоперезапуск** при ошибках
- 🌐 **SSL сертификаты** автоматически
- 📊 **Мониторинг** и логи

## 🔧 **Команды для обновления:**

```bash
# Обновить код
git add .
git commit -m "Обновление бота"

# Railway автоматически обновит бота!
```
